Post-it Website Project
=======================
Guillaume Legrain

This is a test website project trying to build a REST PHP API

## Getting started
### Requirements
- node.js
- npm
- redis-server


### Quickstart
1. Start your redis server

       redis-server

2. start the post-it app

       node app.js

### On Mac OS X
Redis can be installed on Mac OS X using [Homebrew](http://brew.sh/)
      brew install redis
