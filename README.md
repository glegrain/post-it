Post-it Website Project
=======================
Guillaume Legrain

This is a test website project trying to build a REST API.

## Getting started
### Requirements
- node.js
- npm


### Quickstart
1. Install required node modules

```
npm install
```

2. start the post-it app

```
npm start
```
